# Contributing

Thanks for contributing to Audio Converter.

## Development setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
python main.py
```

## Pull request checklist

- Keep source code, comments, and docs in English.
- Do not commit local artifacts such as logs, local settings, virtual environments, or generated output files.
- Keep changes focused and include a clear description.
- Update `README.md` and `CHANGELOG.md` when behavior changes.

## Coding notes

- Preserve existing project structure under `src/convertidor_audio`.
- Keep UI language support intact (English/Spanish in-app strings are expected).
- Prefer small, reviewable commits.
