# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Changed
- Updated UI spacing and alignment in Output settings for better layout on wide windows.
- Improved table corner styling for smoother visual borders.
- Restored bilingual language support (English and Spanish) with English as default.
- Refined setup and requirements documentation.

## [1.0.0] - 2026-04-21

### Added
- Initial public release of Audio Converter.
- PySide6 desktop UI with queue-based per-file conversion settings.
- FFmpeg/FFprobe conversion pipeline with per-file progress and cancellation.
- Persistent preferences for language, colors, output path, and conversion defaults.
